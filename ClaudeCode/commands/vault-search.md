# /obsidian-memory:vault-search

Search your Obsidian Vault for information.

## Usage
```
/obsidian-memory:vault-search <query>
```

## Example
```
/obsidian-memory:vault-search trading system architecture
```

This command searches through all markdown files in your Obsidian vault and returns the most relevant matches.
